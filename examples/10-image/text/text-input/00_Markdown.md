Zum Beispiel passt sich das Buchungsformular deinem Website-Design an, es können weitere Eingabefelder hinzugefügt werden und die Texte in diesem können angepasst werden. Die Rechnungsstellung kann so konfiguriert werden, das Butlerapp genau mit deinen Zahlungsbedingungen in Rechnung stellt. Es können alle Dokumente, die du im Geschäftsbetrieb nutzt hochgeladen werden, und dann mit im System gespeicherten Daten vorausgefüllt werden.

Wie ist Butlerapp mit meiner Buchhaltung verknüpft?
- Butlerapp kann Rechnungen automatisch zu DATEV Online übertragen. Darüber hinaus können Rechnungen monatsweise mit einem Klick heruntergeladen werden.
